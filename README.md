# Task Marketplace v2

**Task Management System for GenLayer SocialFi Ecosystem**

**Contract Address:**  
`0xC5e8C94e80b7D50c4CEA41B0B428dDc28f4C4799`

## Built By
[@cryptofunny724](https://x.com/cryptofunny724)

## Overview
Allows users to create and complete tasks with automatic reputation rewards.

## Features
- `create_task(description)` — Create a new task (+5 reputation)
- `complete_task()` — Complete a task (+20 reputation)
- `my_task()` — View user's current task
- `my_score()` — View reputation score

## Status
Stable and fully functional on GenLayer Bradbury Testnet.
